Как взломать пентагон с домашнего компьютера?

сентябрь 21, 2016 in Category

У Вас бывало так - чего-то хочется, а чего не знаю, но руки чешуться?! Так вот! Как на счет создать несколько простых программок используя команднуюстроку Windows. Что нам для этого понадибиться? Достаточно будет стандартного блокнота. Создаем обычный файл с расширением ".txt". Открываем его и записываем следующие строки...

setlocal enabledelayedexpansion
@echo off
color 0A

- Первые три строчки определяют временное изменение значений переменных среды в командных файлах Windows, скрывают вывод выполняемых команд, задают цвет фона и текста.

:m2
cls
echo Подбор пароля к серверу Пентагона...
echo.
set j=0

- следующие строчки устанавливают метку перехода программы, очищают экран, вывод пустой строки и объявление переменной...

:m1
set line=
for /l %%i in (0,1,36) do (
set /a dig=!random!%%3
if "!dig!"=="2" (set "dig= ")
set line=!line!!dig!
)
echo.%line%
call :wait
set /a j=%j%+1
if %j%==42 (call :pass & call :wait & goto m2)
goto m1

- Смысл предыдующей записи: получить рандомно значение 0, 1 или 2. Если выпола 2, то заменить ее на пробел. Записать в строчку 36 символов (0, 1, " "). Вызвать процедуру "ждать" (для плавного вывода на экран).

:wait
@Echo off
@echo wscript.Sleep 100>"%temp%\sleep30.vbs"
cscript //nologo "%temp%\sleep30.vbs"
del "%temp%\sleep30.vbs"
exit /b

- Процедура "ожидания" собственной персоной))))

:pass
set b=0
echo.

- Счетчик символов в пароле сбрасываем на "0".

:p1
call :wait
< nul set /p "m=* "
set /a b=%b%+1
if %b%==16 (
echo "Alt + 7" ERROR
call :wait
exit /b)
goto p1

- Вывод пароля на экран. Внимание! После вывода "* * * * * * * * * * * * * * * * *" прозвучит сигнал. Для этого при вводе кода необходимо нажать "Alt + 7". Вот собственно и все! Осталось "Сохранить как.." выбрать кодировку 866(ОЕМ-Русская) и потом заменить расшерение файла с ".txt" на ".bat".
Уходя с рабочего места, я запускаю этот файлик как заставку.
